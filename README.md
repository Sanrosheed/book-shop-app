# Full-Stack Online Book Store

A simple full-stack web application that allows users to get Ebooks and display their books. Built with React, Tailwind CSS, Node.js, Express, and MongoDB.

## Features
- Can view all books from the database
- Add new books via a validated form
- State management using Context API
- React Router was used to handle navigation between the pages.
- Error handling and input validation

### Bonus Features
- User Authentication
- Deployed application

## Tech Stack
### Frontend:
- React
- React Router
- Context API / Redux Toolkit
- Tailwind CSS

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB (Mongoose)


## Installation & Setup

### 1. Clone the repository
git clone https://

### 2. Install dependencies

Frontend:
cd Frontend
npm install

Backend:
cd Backend
npm install

### 3. Set up environment variables

Create a `.env` file in the server folder and add:
MONGO_URI
PORT=5000

### 4. Run the app

Backend:
npm run dev

Frontend:
npm start
